# A3-Java-Pedidos
Sistema de pedidos para lanchonete, usando Java e SpringBoot.


Para usar o sistema, é necessário Java JDK 21, e o XAMPP instalado, com um usuário criado chamado 'grupo' e a senha 'BolinhaDeNeve345', e um banco chamado 'lanchonete'. Também é necessário ter o Maven, Python para o servidor web local.

Abrir a pasta do projeto, abrir o terminal e dar __mvn clean install__ . Após isso, clicar em run na IDE. Abrir um terminal na pasta _A3-Java-Pedidos\src\main\resources\static_ e executar __python -m http.server__ .
